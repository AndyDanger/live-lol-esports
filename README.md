# 🔑 Key Differences
* Defaults to dark theme 😎
* Shows patch version 🤓
* Shows series information such as "Best of 5" and "European Semi-Finals" 🏆
* Shows in-game timer (approximate) ⏲
* Show game start times and upcoming games 📅
